# Google-Dork-Cheat-sheet
## Search Filters

# Search Operators

| Operator | Description |
|----------|-------------|
| `""` | Exact phrase search |
| `OR` | Either keyword |
| `AND` | Both keywords |
| `-` | Exclude keyword |
| `*` | Wildcard |
| `()` | Group expressions |
| `site:` | Restrict to a website |
| `filetype:` | Search a file format |
| `ext:` | File extension search |
| `inurl:` | Search URL |
| `allinurl:` | Search all URL keywords |
| `intitle:` | Search page title |
| `allintitle:` | Search all title keywords |
| `intext:` | Search page text |
| `allintext:` | Search all page text keywords |
| `related:` | Similar websites |
| `cache:` | Cached webpage |
| `define:` | Dictionary definition |
| `before:` | Before a date |
| `after:` | After a date |
| `AROUND(X)` | Nearby words |
| `..` | Numeric range |

# Search Filters

| Filter | Description | Example |
|---------|-------------|---------|
| `allintext:` | Searches for all specified keywords in the page body. | `allintext:"machine learning tutorial"` |
| `intext:` | Searches for a keyword anywhere in the page text. | `intext:"artificial intelligence"` |
| `allinurl:` | Searches URLs containing all specified keywords. | `allinurl:python tutorial beginner` |
| `inurl:` | Searches URLs containing a specific keyword. | `inurl:python` |
| `allintitle:` | Searches page titles containing all specified keywords. | `allintitle:python tutorial beginner` |
| `intitle:` | Searches page titles containing a keyword. | `intitle:"Python"` |
| `site:` | Restricts results to a specific website or domain. | `site:github.com markdown` |
| `filetype:` | Searches for a particular file format. | `filetype:pdf astronomy` |
| `ext:` | Alternative to `filetype:` (not officially documented). | `ext:pdf` |
| `related:` | Finds websites similar to another website. | `related:github.com` |
| `cache:` | Displays Google's cached version of a webpage (limited availability). | `cache:example.com` |
| `" "` | Searches for an exact phrase. | `"black hole"` |
| `-` | Excludes a word from search results. | `python -snake` |
| `OR` | Returns results matching either term. | `python OR java` |
| `AND` | Returns results matching both terms (usually implied). | `python AND flask` |
| `*` | Wildcard representing one or more unknown words. | `"largest * in the world"` |
| `()` | Groups multiple search expressions. | `(python OR java) tutorial` |
| `..` | Searches within a numeric range. | `camera $300..$800` |
| `define:` | Displays the definition of a word. | `define:entropy` |
| `AROUND(X)` | Finds words occurring within X words of each other. | `NASA AROUND(3) Artemis` |
| `before:` | Searches results before a specific date. | `AI before:2022-01-01` |
| `after:` | Searches results after a specific date. | `AI after:2024-01-01` |
| `source:` | Restricts results to a news source (Google News). | `NASA source:Reuters` |
| `weather:` | Shows weather information. | `weather:London` |
| `stocks:` | Shows stock information. | `stocks:GOOG` |
| `map:` | Opens Google Maps search. | `map:New York` |
